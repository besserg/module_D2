1. Установить зависимости из requirements.txt
2. Папки success и fail обязательны для работы и должны быть в той же директории, что и bottle_server.py
3. Запустить сервер: python3 bottle_server.py или python bottle_server.py (смотря у кого как настроено)

P.S. Для информации:
Файлы: Procfile, requirements.txt и runtime.txt - необходимы для запуска проекта на Heroku (дополнительно требуется настройка проекта при помощи heroku CLI. См. документацию по Heroku)
